## 📋 启动方式（3种）

### 方式一：双击启动（推荐）⭐
```
双击 start.bat 文件
```
**优点：** 最简单，无需任何命令行操作

### 方式二：PowerShell 启动
```powershell
cd D:\Trae\接口文件
.\venv\Scripts\python.exe app.py
```

### 方式三：CMD 启动
```cmd
cd /d D:\Trae\接口文件
venv\Scripts\python.exe app.py
```

## 🔧 常见问题解决

### 问题1：端口被占用
```powershell
# 查找占用端口的进程
netstat -ano | findstr :9876

# 结束进程（替换 PID）
taskkill /PID <PID号> /F
```

### 问题2：依赖缺失
```powershell
# 重新安装依赖
.\venv\Scripts\pip.exe install -r requirements.txt
```

### 问题3：数据库问题
```powershell
# 删除旧数据库重新初始化
del data\db.sqlite
# 重启服务会自动创建新数据库
```

### 问题4：权限错误
```powershell
# 以管理员身份运行 PowerShell 或 CMD
```

## ⚙️ 可选配置

如需使用 AI 功能和邮件功能，编辑 `.env` 文件：

```bash
# DeepSeek AI (AI对话功能)
DEEPSEEK_API_KEY=你的密钥

# DKFILE API (远程存储)
DKFILE_API_KEY=你的密钥

# SMTP 邮件 (验证码发送)
SMTP_HOST=smtp.qq.com
SMTP_PORT=587
SMTP_USERNAME=你的邮箱
SMTP_PASSWORD=授权码
SMTP_FROM=发件人邮箱
```

**注意：** 不配置这些功能也能正常运行，只是对应功能不可用。
