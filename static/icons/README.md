# PWA图标说明

本目录用于存放PWA（渐进式Web应用）所需的图标文件。

## 所需图标

请添加以下尺寸的PNG图标到本目录：

1. **192x192像素** - 用于普通屏幕设备
   - 文件名：`icon-192x192.png`
   - 推荐尺寸：192px × 192px

2. **512x512像素** - 用于高分辨率屏幕设备
   - 文件名：`icon-512x512.png`
   - 推荐尺寸：512px × 512px

## 图标要求

- 格式：PNG
- 背景：建议使用透明背景
- 内容：清晰的应用Logo
- 比例：正方形（1:1）

## 生成图标

您可以使用在线工具生成PWA所需的所有图标尺寸，例如：
- [PWA Asset Generator](https://www.pwabuilder.com/imageGenerator)
- [RealFaviconGenerator](https://realfavicongenerator.net/)

这些工具可以从一个源图像生成所有必要尺寸的图标，并提供完整的配置代码。

## 注意事项

- 确保图标文件命名正确，否则PWA功能可能无法正常工作
- 图标应清晰可见，建议使用简单、识别度高的设计
- 定期更新图标，以保持与应用品牌一致
