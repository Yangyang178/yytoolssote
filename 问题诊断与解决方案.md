## 🔧 故障排除

### 如果双击无法启动：

#### 方法 A：右键管理员运行
1. 右键点击 `一键启动.bat`
2. 选择 "以管理员身份运行"

#### 方法 B：手动启动
1. 打开 CMD 或 PowerShell
2. 输入以下命令：
```cmd
cd /d D:\Trae\接口文件
venv\Scripts\python.exe app.py
```

### 如果端口 9876 被占用：
```powershell
# 查找占用进程
netstat -ano | findstr :9876

# 结束进程（替换 PID）
taskkill /PID <PID号> /F
```

### 如果提示模块缺失：
```powershell
# 重新安装所有依赖
.\venv\Scripts\pip.exe install -r requirements.txt --upgrade
```

### 如果数据库损坏：
```powershell
# 删除数据库重新初始化
del D:\Trae\接口文件\data\db.sqlite
# 重启服务会自动创建新数据库
```

## 📊 性能优化建议

### 1. 减少启动时间（可选）
编辑 `app.py` 底部，将 debug 改为 False：
```python
app.run(debug=False, host='0.0.0.0', port=9876)
```
**优点：** 启动更快，适合生产环境

### 2. 启用多线程（高并发）
```python
app.run(debug=True, host='0.0.0.0', port=9876, threaded=True)
```
**优点：** 支持同时处理多个请求

### 3. 使用生产服务器（部署时）
安装 gunicorn 或 waitress：
```bash
pip install waitress
waitress-serve --port=9876 app:app
```
